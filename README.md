# ChromeDriver Bug Repro Templates

This template repository has a collection of ChromeDriver regression test templates
for different operating system and technology combinations.

## Using this repo

1.  Click the "Use this template" button and select "Create a new repository."
2.  Select an owner for the new repository, and enter a repository name.
3.  Choose the visibility of the new repository (Public or Private).
4.  Click "Create repository from template."
5.  In the newly created repository, go to the "Settings" tab.
6.  In the left sidebar, click "Actions," then click "General."
7.  Under "Workflow permissions," select "Read and write permissions." This is necessary for the GitHub Actions in this repository to have sufficient permissions to run.
8.  Clone the newly created repository to your local machine.
9.  Follow the instructions in the README.md file of the specific template you want to use to reproduce a bug.
